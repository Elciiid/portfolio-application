# Jonas David Portfolio

Personal portfolio for Jonas David, a full-stack developer focused on production internal systems, operational tooling, and AI-assisted automation.

## Overview

The site presents shipped systems from La Rose Noire Philippines alongside personal AI/security and automation projects. The goal is to make the work easy for recruiters and technical reviewers to scan: what each system does, why it mattered, and what engineering constraints were handled.

## Tech Stack

- Next.js 15 App Router
- React 18
- TypeScript
- Tailwind CSS 4
- Framer Motion
- Lucide React
- Resend contact API

## Featured Systems

- MP3 Streamer: synchronized internal audio broadcasting across factory zones.
- ITicketHub: role-based IT ticket triage with audit history and assignments.
- CentralPoint Portal: intranet surface for internal app access and live feeds.
- CentralPoint Core: admin layer for messaging, scheduling, venue conflict checks, and permissions.
- Facebook Phishing Detector: applied AI/security browser extension using Autoencoder and GCN-based detection.
- AI Web3 Automation Assistant: exploratory event-driven automation for wallet monitoring, summaries, and Telegram alerts.

## Local Setup

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

For production checks:

```bash
npm run lint
npm run build
```

## Project Philosophy

This portfolio emphasizes practical systems work over visual-only demos: role-based access, auditability, synchronization, approval flows, internal dashboards, and automation workflows. Personal projects are framed as exploration into applied AI, security, and event-driven automation.
