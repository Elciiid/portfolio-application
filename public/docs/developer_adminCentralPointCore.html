2. Authentication Flow

CentralPoint uses a session-based authentication system linked to the company master list.

                    sequenceDiagram
                        participant User
                        participant Login as login.php
                        participant DB as MSSQL (LRNPH_E)
                        
                        User->>Login: Username + Password
                        Login->>DB: Verify BiometricsID & Password Hash
                        DB-->>Login: User Records (ID, Dept, Name)

                        alt Success
                            Login->>Login: Start Session & Set $_SESSION
                            Login-->>User: Redirect to index.php
                        else Failure
                            Login-->>User: Show Auth Error
                        end

4. ChatNow (Messaging)

A real-time messaging component capable of DMs, Group Chats, and "Stories/Notes" functionality.

Database Context: LRNPH_ITmanagement (Remote: 10.2.0.9)

Key Features:

  • Stories/Notes: 24-hour temporary status updates with reaction support (UserNotes).
  • Unified Search: Searches both individual users and group participants via STRING_AGG.
  • Rich APIs: chatnow_api.php handles async message sending, story reactions, and read status updates.

9. Developer Tasks

How to Add a New Admin Feature

  1. Create components/new_feature.php.
  2. Add a case in admin.php to include the new file.
  3. Add the feature to portal_AppModules via the System Settings UI.
  4. Assign permissions in User Management.
Important: Always use sqlsrv_prepare or sqlsrv_query with parameters for external user inputs to prevent SQL injection.
